# Exercícios – Introdução à Ciência de Dados

Este repositório foi criado para armazenar os exercícios propostos nas aulas da disciplina de Introdução à Ciência de Dados da minha graduação.

O objetivo é reunir os notebooks e projetos desenvolvidos ao longo da disciplina, servindo como registro de aprendizado, prática e evolução no estudo de ciência de dados.

**🎯 Objetivo do Repositório**

Organizar os exercícios realizados durante a disciplina

Praticar conceitos fundamentais de Ciência de Dados

Aplicar técnicas de análise e manipulação de dados em Python

Manter um histórico dos projetos desenvolvidos ao longo do curso

**📖 Contexto Acadêmico**

Este repositório faz parte das atividades da disciplina de Introdução à Ciência de Dados da minha graduação.

Os materiais aqui presentes têm fins educacionais.
